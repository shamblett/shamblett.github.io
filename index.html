Hi all Dartisans, this my starter site for what will hopefully become a fuller dive into the Dartiverse.

I'm user 'shamblett' in google groups etc, github, medium and on pub where you can find my published packages.

I've been in the Dartiverse since the start of Dart 1.0, my first published package, xml2json, was back in September 2013.

I've seen a few changes here and there and picked up a lot of Dart knowledge along the way.

I mainly stay on the server side of things in the comfort of the VM though I have a few both server and web packages. I've not yet had a look at Flutter, mainly because I don't really do web/app development but it is becoming insanely popular so I might one day have a dabble.

The advent of dart 2.6 has really changed the game for me as it brings two things I sorely needed, FFI and Dart2Native.

My latest package, mraa, couldn't exist without FFI and wouldn't really be deployable without Dart2Native.

Using these tools I can wrap any native library of my choice and when testing is complete just turn the Dart program into a native exe for deployment to the platform, if this isn't a killer feature I don't know what is!
